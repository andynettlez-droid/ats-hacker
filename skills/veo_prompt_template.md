# Veo Prompt Template

Use Veo/Gemini for short 9:16 clips only when the shot benefits from cinematic movement. Product UI and resume text should usually be edited deterministically with ffmpeg/Remotion so it stays readable.

## Default Clip Contract

- Aspect ratio: 9:16.
- Duration: 8 seconds unless the API/model allows otherwise.
- No on-screen text in generated video. Add captions and labels in edit.
- Avoid unreadable fake resume text when the audience needs to inspect details.
- Keep movement simple and motivated.
- Use Abby reference image for human/recruiter presence.
- Use Signal/orb reference image for mascot consistency.
