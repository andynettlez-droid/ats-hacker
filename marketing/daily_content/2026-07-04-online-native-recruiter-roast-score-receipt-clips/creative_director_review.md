## Batch Notes

This batch is materially better than the prior HubSpot-style scripts because it shows the receipt before the payoff. The remaining weakness is role variety: all three clips use the same data analyst case. That is acceptable for testing visual styles, but the next daily batch should rotate roles across software, nursing, sales, project management, and operations.

The pipeline should continue rejecting:
- Generic HubSpot/CAC examples unless the selected role is actually marketing.
- Random score jumps without visible score basis.
- Reused openings across a daily batch.
- Narration that explains ATS in abstract terms instead of showing a resume/JD conflict.
