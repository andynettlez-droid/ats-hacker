# Veo Prompt Template

Use Veo/Gemini for short cinematic 9:16 clips only when motion adds value. Keep product UI, captions, labels, and readable resume/JD text in the edit layer, not inside Veo.

## Clip Contract

- Format: vertical 9:16.
- Duration: 8 seconds unless the model/API allows a different explicit duration.
- Match the reference style first. For resume-review clips, use natural desk/tablet lighting and simple social captions, not a forced Signal palette.
- No on-screen text generated by Veo.
- No fake UI text or unreadable fake resume text when the audience needs to inspect the artifact.
- Use Abby reference image for recruiter/human presence when needed.
- Do not use Signal/orb reference images unless the brief is explicitly an animated Signal mascot concept.
- Smooth camera movement. Avoid glitch flicker.
- If the output has a visible provider watermark, do not use it for public posting unless the source license/export allows it.
